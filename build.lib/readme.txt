Store copies for gp-module.jar library.
    gp-modules.jar, the version you get when you download the GP Client library from a GP 3.4.0 server.
        The source code is buried in an old rev in SVN. Since this jar was created, the library has been
        moved to a different base package, (from org.genepattern.* to org.broadinstitute.*).
        
    gp-modules-threadsafe-rev27666-2012-09-17.jar, a modified (to be thread-safe) version of gp-modules.jar, built
        from latest (circa Sept 17, 2012) rev in SVN. Base package is 'org.broadinstitute.*'.
